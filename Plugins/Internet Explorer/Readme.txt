Para poder compilar este proyecto se necesita tener instalado NSIS,
obtenible desde http://nsis.sourceforge.net/Main_Page.

Para la edición de los ficheros ICO hace falta algún programa que permita ICOs
multi imagen, como @icon_sushi, obtenible en http://www.towofu.net/soft/e-index.php

Para la edición del resto de ficheros basta con un editor de textos, como el 
Bloc de Notas o JEdit.
